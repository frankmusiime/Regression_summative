# Predicting University Performance Scores Using Academic and Research Indicators

## Mission and Problem Statement

Universities contribute significantly to education quality, research development, innovation, and economic growth. However, identifying the factors that influence university performance can help institutions make better strategic decisions.

This project develops a machine learning regression system that predicts a university's overall performance score using academic, research, faculty, citation, graduate outcome, and innovation indicators.

The objective is to train and deploy a regression model capable of predicting university performance scores through a FastAPI backend and a Flutter mobile application.

# Dataset Description

## Dataset Source

The dataset used in this project is the **World University Ranking Dataset** obtained from Kaggle.

Dataset Link:https://www.kaggle.com/datasets/mylesoneill/world-university-rankings?select=cwurData.csv

## Dataset Overview

The dataset contains university ranking information collected across different countries and years.

Each record represents a university's performance for a particular ranking year and contains academic, research, faculty, influence, citation, and innovation indicators.

| Dataset Property | Description |
|---|---|
| Source | Kaggle |
| Dataset Type | Regression Dataset |
| Number of Records | 2,200 university records |
| Number of Features | 14 variables |
| Target Variable | score |
| Prediction Objective | Predict university performance score |

# Dataset Features

The dataset contains the following variables:

| Feature | Description |
|---|---|
| world_rank | Global university ranking |
| institution | University name |
| country | University country |
| national_rank | University ranking within country |
| quality_of_education | Education quality indicator |
| alumni_employment | Graduate employment indicator |
| quality_of_faculty | Faculty quality indicator |
| publications | Research publication indicator |
| influence | Academic influence indicator |
| citations | Research citation indicator |
| broad_impact | Overall research impact |
| patents | Innovation indicator |
| score | Overall university performance score (Target) |
| year | Ranking year |


# Target Variable

The model predicts:

```
score
```

The score represents the overall university performance based on education quality, research output, faculty quality, citations, influence, and innovation indicators.

A higher score indicates stronger university performance.

The SGDRegressor model was optimized using gradient descent.

The training process included:

- Splitting data into training and testing sets.
- Scaling features.
- Training multiple regression algorithms.
- Comparing model losses.
- Selecting the best-performing model.


# Model Evaluation

The models were evaluated using:

| Metric | Description |
|---|---|
| MAE | Average absolute prediction error |
| MSE | Average squared prediction error |
| RMSE | Prediction error in target variable units |
| R² Score | Percentage of variation explained by the model |

The main loss metric used for comparison was:

```
Mean Squared Error (MSE)
```

The model with the lowest loss was selected as the final deployed model.
